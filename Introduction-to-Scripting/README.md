# 🧠 Introduction to Scripting

Three programs built from original course design documents (pseudocode and, for the text adventure, a full room map and item layout).

## 💰 payroll_calculator.py

Calculates weekly pay from hours worked, applying time and a half overtime for anything past 40 hours.

```
python payroll_calculator.py
```

## 🎯 higher_lower_game.py

Player sets a number range, the game picks a secret number inside it, and the player guesses until they land on it, with "too high" or "too low" feedback after each guess.

```
python higher_lower_game.py
```

## 🖥️ data_center_breach.py

A text adventure. You're inside a compromised data center after a cyberattack. Move through the facility (`go north`, `go south`, `go east`, `go west`), collect five items (`get network diagram`, etc.), and reach the Server Room with everything in hand to stop the breach. Entering the Server Room without all five items ends the game.

```
python data_center_breach.py
```

## 📋 Status

All three implemented and tested against the original course design. This is a from scratch build based on my own pseudocode and map layout from the course, since the original submissions were design documents rather than runnable code.

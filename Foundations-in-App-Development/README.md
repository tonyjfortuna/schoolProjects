# 🐾 Foundations in App Development

A pet check in system for a boarding facility, built from the original course pseudocode.

## 🏠 pet_checkin.py

Checks a dog or cat into an open boarding spot. Dogs and cats have different data requirements, dogs need weight and a grooming decision if staying two or more nights, cats just need name, age, and length of stay. Reports "no space available" if every spot for that pet type is full.

```
python pet_checkin.py
```

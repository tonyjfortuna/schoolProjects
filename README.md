# 📚 schoolProjects

Coursework from my B.S. in Software Engineering at SNHU. Each folder covers a specific course and includes working code, not just write ups.

## 🗂️ Contents

- **Introduction-to-Scripting**: A payroll calculator, a higher/lower guessing game, and a text adventure game built around a data center breach scenario.
- **Foundations-in-App-Development**: A pet check in system for a boarding facility, handling dogs and cats with different data requirements.
- **Applied-Statistics-for-STEM**: Three Jupyter notebooks covering confidence intervals, hypothesis testing, and regression, using NBA team performance data.

## 📝 A note on Introduction to Scripting and Foundations in App Development

The original submissions for these two courses were design documents: pseudocode and flowcharts describing how each program should work. This repo contains working Python implementations built from those original designs, not new assignments. See each folder's README for specifics.

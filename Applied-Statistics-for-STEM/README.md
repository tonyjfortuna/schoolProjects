# 📊 Applied Statistics for STEM

Three Jupyter notebooks analyzing NBA team performance using FiveThirtyEight's NBA Elo dataset.

## 📓 Notebooks

- **Project_One.ipynb**: Descriptive statistics and confidence intervals comparing points scored between two NBA teams, home versus away performance, and relative skill rating (Elo).
- **Project_Two.ipynb**: Hypothesis testing, including a two proportion z-test comparing win rates and comparisons of mean relative skill and scoring between teams across different eras.
- **Project_Three.ipynb**: Simple and multiple linear regression, modeling total wins against average relative skill and average points scored.

## ▶️ Running these

These notebooks were built against FiveThirtyEight's `nbaallelo.csv` dataset. To run them, download that dataset and place it in this folder before opening the notebooks in Jupyter.
